# Lan Chi Personal Creative Space

Một trang web cá nhân nhỏ bằng Streamlit cho Lan Chi: có khu chữa lành, lịch công việc, chuẩn bị tài sản, nghỉ ngơi, cinema wall, moodboard ảnh và ghi chú hôm nay.

## Cách chạy trên máy

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Cách đưa lên Streamlit Community Cloud

1. Tạo một repository trên GitHub.
2. Upload 3 file này lên repository:
   - `app.py`
   - `requirements.txt`
   - `README.md`
3. Vào Streamlit Community Cloud.
4. Chọn repository vừa tạo.
5. Chọn file chạy chính là `app.py`.
6. Deploy.

## Ghi chú

Phần upload ảnh và ghi chú trong bản này chỉ hiện trong phiên đang mở. Khi tải lại trang, dữ liệu chưa được lưu lâu dài.
